# Mi Primera Página Web

Una página web simple con una galería de imágenes y una lista de recursos para encontrar imágenes de alta calidad.

## Características

- 📸 Galería de imágenes responsiva
- 🎨 Diseño moderno con gradientes
- 📱 Compatible con dispositivos móviles
- ✨ Efectos hover en las imágenes

## Recursos

- [Openverse](https://openverse.org/)
- [Unsplash](https://www.unsplash.com)
- [Pixabay](https://pixabay.com/)

## Cómo ver la página

Esta página está publicada en GitHub Pages. Accede al enlace en la sección de "About" del repositorio.
